# 端侧AI - Aura 慢病健康助手
- 项目简介：Aura 是一款专为慢性病患者设计的智能健康助手，致力于将繁琐的健康管理转化为简单、安心的日常陪伴。
- 运行平台：基于英特尔AI-PC，通过本地化部署通义千问（Qwen）大模型，确保所有个人健康信息在用户设备端处理。
- 详细内容见 modelscope ：https://modelscope.cn/learn/2920

# Edge-side AI - Aura Chronic Disease Health Assistant 
- Project Overview: Aura is an intelligent health assistant specifically designed for patients with chronic diseases, aiming to transform complicated health management into simple and reassuring daily companionship.
- Operating Platform: Based on Intel AI-PC, deploying the Qwen large model locally to ensure all personal health information is processed on the user's device.
- For more details, visit modelscope: https://modelscope.cn/learn/2920
